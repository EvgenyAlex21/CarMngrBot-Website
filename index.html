Информация о боте

CAR MANAGER (@CarMngrBot) — это мессенджерный многофункциональный Telegram бот для автомобилистов, который упрощает повседневные задачи, связанные с использованием и обслуживанием автомобиля. Бот создан для повышения удобства планирования поездок, учета расходов и ремонтов, диагностики транспорта и получения актуальной информации о погоде, ценах на топливо, маршрутах и других важных аспектах.

Основные функции

  • Расчет топлива: бот помогает рассчитать количество топлива для поездки с учетом расстояния, расхода топлива транспорта, цены на топливо и количества пассажиров. Также доступен просмотр и удаление ранее сохранённых поездок.
  • Ваш транспорт: добавляйте данные о вашем транспорте (марка, модель, год выпуска, госномер РФ), удаляйте ненужные записи и просматривайте список сохранённых транспортов.
  • Траты и ремонт: фиксируйте расходы на транспорт и ремонты по категориям, анализируйте данные, просматривайте и удаляйте записи.
  • Поиск транспорта: сохраните геопозицию вашего транспорта и получите ссылку на маршрут к вашему текущему местоположению. Это поможет, если вы забыли, где припарковались.
  • Поиск мест: найдите ближайшие интересующие вас места на основе вашей текущей геопозиции, например, авто заправочные станции, автомойки, автосервисы и так далее.
  • Погода: отправьте свою геопозицию, чтобы получить прогноз погоды на сегодня, завтра, неделю или месяц. Это удобно для планирования дальних поездок.
  • Цены на топливо: бот предоставляет актуальную информацию о стоимости различных видов топлива (АИ-92, АИ-95, АИ-98, АИ-100, ДТ, ГАЗ) в вашем городе.
  • Код региона: определите регион транспортного средства по госномеру РФ или узнайте региона по его коду.
  • Анти-радар: бот может отображать камеры видеофиксации нарушений ПДД, предупреждая о возможных штрафах. Геопозиция обновляется вручную.
  • Напоминания: создавайте напоминания о важных задачах или событиях, связанных с автомобилем, указывая дату, время и текст. Можно просматривать и удалять напоминания.
  • Коды ошибок OBD2: поддерживает диагностику автомобиля через поиск и расшифровку кодов ошибок OBD2. Удобно для владельцев, желающих самостоятельно понять причину неисправности.

Инструкция по использованию бота

1. Как начать использовать бота

Чтобы начать использовать бота, отправьте команду /start. Это откроет меню с доступными действиями. После активации бота следуйте текстовым инструкциям для выполнения нужных действий.

2. Описание функций

2.1. Расчет топлива и сохранение поездок

Для расчета расхода топлива выберите пункт «Расчет топлива» в меню. Затем отправьте следующие данные:

  • Начальное местоположение: укажите ваше начальное местоположение.
  • Конечное местоположение: укажите конечное местоположение.
  • Дата поездки: укажите дату поездки в формате ДД.ММ.ГГГГ.
  • Расстояние: укажите расстояние поездки в километрах.
  • Тип топлива: укажите тип топлива (например, АИ-92, ДТ, ГАЗ).
  • Цена топлива за литр: укажите цену за литр топлива в рублях.
  • Расход топлива на 100 км: укажите расход топлива вашего автомобиля на 100 км.
  • Количество пассажиров: укажите количество пассажиров в автомобиле.

После того как вы отправите все данные, бот произведет расчет и предоставит вам детализированный отчет о расходах на топливо для вашей поездки. Также у вас будет возможность сохранить поездку для дальнейшего просмотра или анализа.

Для просмотра сохраненных поездок выберите пункт «Просмотр поездок» в меню.

Если вы хотите удалить сохраненную поездку, выберите пункт «Удалить поездку» и следуйте инструкциям бота.

2.2. Ваш транспорт

Чтобы управлять транспортом, выберите пункт «Ваш транспорт» в меню. В этом разделе доступны следующие функции:

  • Добавить транспорт: Введите данные вашего транспорта:
    • Марка: укажите марку транспорта.
    • Модель: укажите модель транспорта.
    • Год выпуска: укажите год выпуска транспорта.
    • Госномер: введите государственный регистрационный номер в формате РФ.
  • Удалить транспорт: Выберите транспорт, который вы хотите удалить, из списка сохраненных.
  • Посмотреть транспорт: Получите список всех добавленных транспортных средств с их основными характеристиками.

После добавления транспорта он станет доступным для использования в других функциях бота, таких как траты и ремонты.

2.3. Траты и ремонты

Чтобы записать трату или ремонт, выберите соответствующую команду в меню:

  • Для записи новой траты используйте команду «Записать трату».
  • Для записи нового ремонта используйте команду «Записать ремонт».

Введите следующие данные:

  • Транспорт — выберите транспорт для записи.
  • Категория — выберите категорию для записи (вы можете добавить свою категорию или удалить ее).
  • Название траты или ремонта.
  • Дата в формате ДД.ММ.ГГГГ.
  • Сумма траты или ремонта.
  • Описание для траты или ремонта.

Чтобы просмотреть свои траты и ремонты по конкретному транспорту, выберите команду «Посмотреть траты» или «Посмотреть ремонты». Вы можете выбрать фильтрацию по:

  • «По категории» — для просмотра по конкретной категории.
  • «За месяц» — для просмотра записей за конкретный месяц.
  • «За год» — для записей за конкретный год.
  • «За всё время» — для всех записей.

Чтобы удалить свои траты и ремонты по конкретному транспорту, выберите команду «Удалить траты» или «Удалить ремонты». Затем выберите:

  • «По категории» — для удаления по конкетной категории.
  • «За месяц» — для удаления записей за определенный месяц.
  • «За год» — для удаления записей за год.
  • «За всё время» — для удаления всех записей.

2.4. Поиск транспорта

Для поиска транспорта выберите команду «Найти транспорт» и укажите следующие данные:

  • Геопозиция транспорта: отправьте местоположение транспорта.
  • Ваша геопозиция: отправьте свое текущее местоположение.

Бот предоставит вам ссылку на маршрут для поиска.

2.5. Поиск мест

Для поиска ближайших мест выберите команду «Поиск мест» и выполните следующие действия:

  • Категория: выберите категорию для поиска.
  • Геопозиция: отправьте свою текущую геопозицию.

Бот предложит вам список ближайших мест и выдаст ссылки с маршрутом на основе вашей локации.

2.6. Погода

Чтобы узнать прогноз погоды, выберите команду «Погода» и выполните следующие действия:

  • Геопозиция: отправьте свою текущую геопозицию.
  • Период прогноза: выберите один из доступных вариантов:
    • На сегодня.
    • На завтра.
    • На неделю.
    • На месяц.

Бот предоставит прогноз погоды на выбранный период.

2.7. Цены на топливо

Для получения актуальных цен на топливо выберите команду «Цены на топливо» и выполните следующие действия:

  • Город: укажите название вашего города.
  • Тип топлива: выберите один из доступных вариантов:
    • АИ-92.
    • АИ-95.
    • АИ-98.
    • АИ-100.
    • ДТ.
    • ГАЗ.

Бот предоставит актуальные цены на выбранное топливо в вашем городе.

2.8. Код региона

Для получения информации о регионе по госномеру транспортного средства выберите команду «Код региона» и укажите следующие данные:

  • Государственный номер: введите номер автомобиля в формате РФ.
  • Регион: укажите номер региона.

Бот предоставит информацию о транспорте или регионе на основе введенных данных.

2.9. Анти-радар

Для получения информации о камерах видеонаблюдения, фиксирующих нарушения ПДД, выберите команду «Анти-радар» и выполните следующие действия:

  • Геопозиция: отправьте свою текущую геопозицию.

Обновление геопозиции происходит вручную. Бот покажет ближайшие камеры на основе вашей геопозиции.

2.10. Напоминания

Чтобы установить напоминание, выберите команду «Напоминания» и укажите следующие данные:

  • Название: введите название напоминания.
  • Дата: укажите дату напоминания.
  • Время: укажите время напоминания.

Бот отправит уведомление в установленное время. Вы также можете просматривать и удалять напоминания через меню.

2.11. Коды ошибок OBD2

Для диагностики автомобиля по кодам ошибок OBD2 выберите команду «Коды OBD2» и укажите следующие данные:

  • Код ошибки: введите код ошибки, полученный через диагностический прибор.

Бот предоставит расшифровку введенного кода.
