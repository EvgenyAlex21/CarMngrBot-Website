Информация о боте

CAR MANAGER BOT (@CarMngrBot) — это мессенджерный многофункциональный Telegram бот для автомобилистов, который упрощает повседневные задачи, связанные с использованием и обслуживанием автомобиля.

Бот создан для повышения удобства планирования поездок, учета расходов и ремонтов, диагностики транспорта и получения актуальной информации о погоде, ценах на топливо, маршрутах и других важных аспектах.

Начать использование

Основные функции

Подписка на бота

Управление подписками и доступом к функциям бота. Поддержка бесплатных и платных подписок, пробных периодов и реферальной системы.

Инструкция по подписке:

  1. Отправить команду /start
  2. Подписаться на Telegram канал
  3. Получить 3 дня пробного периода (если по реферальной ссылке - +1 день)
  4. Выбрать нужный тип подписки
  5. Приобрести подписку через Telegram Payments или Paymaster
  6. Просматривать статус подписки и историю
  7. Управлять баллами и промокодами
  8. Участвовать в реферальной программе
  9. Получать бонусы за подписку на рекламные каналы

Доступные функции подписки:

  • Просмотр текущей подписки
  • История подписок
  • Отмена подписки
  • Магазин баллов и времени
  • Система промокодов
  • Реферальная система с бонусами

Калькуляторы

Доступ к различным автомобильным калькуляторам для расчетов и планирования.

Инструкция по использованию калькуляторов:

  1. В главном меню выберите "Калькуляторы"
  2. Выберите нужный калькулятор из списка:
    • Расход топлива
    • Алкоголь
    • Автокредит
    • Налог
    • Растаможка
    • ОСАГО
    • Шины
  3. Для каждого калькулятора следуйте инструкциям в соответствующем разделе

Траты и ремонты

Учет расходов на транспорт и ремонты по категориям с возможностью анализа.

Инструкция по учету трат и ремонтов:

  1. Для записи новой траты используйте команду «Записать трату»
  2. Для записи нового ремонта используйте команду «Записать ремонт»
  3. Введите данные:
    • Выберите транспорт
    • Укажите категорию (можно добавить свою)
    • Введите название траты/ремонта
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите сумму
    • Добавьте описание (опционально)
  4. Для просмотра:
    • Выберите «Посмотреть траты» или «Посмотреть ремонты»
    • Можно фильтровать по категории, месяцу, году или за все время
  5. Для удаления:
    • Выберите «Удалить траты» или «Удалить ремонты»
    • Выберите критерий удаления (по категории, за месяц, за год или все)

Найти транспорт

Помощь в поиске маршрута от текущей геопозиции до местоположения транспорта с помощью Яндекс.Карт.

Инструкция по использованию:

  1. В главном меню выберите "Найти транспорт"
  2. Если процесс был прерван, выберите "Продолжить" или "Начать заново"
  3. Отправьте геопозицию транспорта
  4. Отправьте свою геопозицию
  5. Получите ссылку на маршрут в Яндекс.Картах

Поиск мест

Поиск ближайших мест (АЗС, автомойки, автосервисы, парковки, эвакуация, ГИБДД, комиссары, штрафстоянка) по вашей геопозиции с помощью Яндекс.Карт.

Инструкция по использованию:

  1. В главном меню выберите "Поиск мест"
  2. Выберите категорию для поиска
  3. Отправьте свою геопозицию
  4. Получите адрес и ссылку на Яндекс.Карты с ближайшими местами по данной категории

Погода

Получение текущей погоды и прогноза по геопозиции или названию города с использованием нескольких погодных API.

Инструкция по использованию:

  1. В главном меню выберите "Погода"
  2. Отправьте геопозицию или введите название города
  3. Выберите период:
    • Сегодня
    • Завтра
    • Неделя
    • Месяц
  4. Получите данные о погоде за выбранный период

Цены на топливо

Узнайте средние цены на топливо в выбранном городе с возможностью выбора по геопозиции или названию города.

Инструкция по использованию:

  1. В главном меню выберите "Цены на топливо"
  2. Введите город, выберите из последних или отправьте геопозицию
  3. Выберите тип топлива:
    • АИ-92
    • АИ-95
    • АИ-98
    • АИ-100
    • ДТ
    • Газ
  4. Получите список средних цен по брендам АЗС (стандарт и премиум)

Код региона

Определение региона России по коду региона или государственному номеру автомобиля с возможностью проверки авто через AvtoCod.

Инструкция по использованию:

  1. В главном меню выберите "Код региона"
  2. Введите код региона РФ или госномер автомобиля РФ
  3. Получите название региона. Для номеров + ссылка на AvtoCod с проверкой авто

Коды OBD2

Диагностика автомобиля через поиск и расшифровку кодов ошибок OBD2.

Инструкция по диагностике OBD2:

  1. Выберите команду «Коды OBD2»
  2. Введите код ошибки, полученный через диагностический прибор
  3. Бот предоставит расшифровку введенного кода

Напоминания

Создание, просмотр и удаление напоминаний с различными типами повторений (однократные, ежедневные, еженедельные, ежемесячные).

Инструкция по использованию:

  1. В главном меню выберите "Напоминания"
  2. Добавление напоминаний:
    • Введите название
    • Выберите тип повторения (однократное, ежедневное, еженедельное, ежемесячное)
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите время в формате ЧЧ:ММ
  3. Просмотр напоминаний:
    • Выберите статус (активные или истекшие)
    • Выберите тип напоминания для показа
  4. Удаление напоминаний:
    • Выберите статус (активные или истекшие)
    • Выберите тип напоминания
    • Укажите номера напоминаний из списка для удаления

Антирадар

Отслеживание дорожных камер и уведомление о приближении к ним на основе геопозиции пользователя.

Инструкция по использованию:

  1. В главном меню выберите "Антирадар"
  2. Отправьте геопозицию для активации антирадара (обновление геопозиции происходит вручную из-за ограничений на стороне Telegram)
  3. Получайте уведомления о камерах
  4. Для выключения антирадара используйте кнопку "Выключить антирадар"

Прочее

Объединяет функции: новости, курсы валют, уведомления, реклама и чат с администратором.

Инструкция по использованию:

  1. В главном меню выберите "Прочее"
  2. В меню "Прочее" выберите нужную функцию:
    • Новости: выберите количество новостей (3–15) для просмотра
    • Курсы валют: просмотрите актуальные курсы валют
    • Уведомления: включите или выключите уведомления на погоду, цены, курсы или на все сразу
    • Реклама:
      • "Заявка на рекламу": введите тему, даты, время, текст, медиа
      • "Ваши заявки": просмотрите или отмените заявки
    • Чат с админом: свяжитесь через указанный контакт

Подписка на бота

Купить подписку

Выбор периода подписки и оплата через Telegram Payments или Paymaster.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Купить подписку"
  2. Выбрать период через кнопки
  3. Оплатить через Telegram Payments и Paymaster
  4. Получить подтверждение оплаты и бонусы (баллы, скидки)

Ваши подписки

Общий модуль для просмотра активных и истекших подписок.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Ваши подписки"
  2. В меню "Ваши подписки" выберите "Посмотреть подписки" или "История подписок"

Посмотреть подписку

Просмотр активных подписок, их сроков и стоимости.

Инструкция по использованию:

  1. В меню "Ваши подписки" выбрать "Посмотреть подписки"
  2. Просмотреть список активных подписок и итоговую оценку

История подписок

Просмотр истекших подписок с датами и стоимостью.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Ваши подписки".
  2. В меню "Ваши подписки" выбрать "История подписок"
  3. Просмотреть истекшие подписки

Возврат

Отмена подписок и просмотр истории возвратов.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Возврат"
  2. В меню "Возврат" выберите "Отмена подписок" или "История возвратов"

Отмена подписок

Отмена активных подписок с возвратом средств.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Возврат".
  2. В меню "Возврат" выберите "Отменить подписки".
  3. Просмотрите список активных подписок, доступных для возврата (указаны тип, оставшееся время, стоимость, возможный возврат).
  4. Введите номера подписок для отмены.
  5. Подтвердите отмену, выбрав "Подтвердить". Вы получите информацию о сумме возврата и комиссии.
  6. Ожидайте зачисления средств на ваш счет в течение 1–7 рабочих дней. Если возврат не выполнен, обратитесь в поддержку.
  7. Если даты других подписок изменились, вы получите уведомление с новым расписанием.

История возвратов

Просмотр истории возвратов с подробной информацией.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Возврат"
  2. В меню "Возврат" выберите "История возвратов"
  3. Просмотрите список всех возвратов с подробной информацией (сумма, дата, статус, ошибки)
  4. При изменении статуса возврата вы получите уведомление (например, "Возврат успешен" или "Ошибка, обратитесь в поддержку")

Магазин

Покупка баллов и времени подписки.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Магазин"
  2. Перейти в "Пакеты баллов" или "Пакеты времени"
  3. Выбрать товар, оплатить, получить баллы/время

Баллы

Просмотр и обмен баллов на время, скидки или функции.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Баллы"
  2. В меню "Баллы" выбрать "Ваши баллы", "Обменять баллы", "Подарки"
  3. Ваши баллы: возможность просмотреть текущие баллы и историю
  4. Обменять баллы: выбрать тип обмена (время/скидка/функция), далее ввести количество баллов и подтвердить обмен
  5. В меню "Баллы" выбрать "Подарки". Далее см. в раздел "Подарки"

Обменять баллы

Обмен баллов на время, скидки или функции.

Инструкция по использованию:

  1. В меню "Баллы" выбрать "Обменять баллы"
  2. Выбрать тип обмена (время/скидка/функция)
  3. Ввести количество баллов для обмена
  4. Подтвердить обмен

Подарки

Дарение баллов или времени другим пользователям.

Инструкция по использованию:

  1. В меню "Баллы" выбрать "Подарки"
  2. Подарить баллы: ввести @username или id пользователя, кому хотите подарить, далее указать количество баллов
  3. Подарить время: ввести @username или id пользователя, кому хотите подарить, далее выбрать единицу времени и количество
  4. История подарков: просмотреть список подарков

Промокоды

Активация промокодов для получения скидок.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Промокоды"
  2. В меню "Промокоды" выбрать "Ввести промокод", "Ваши промокоды"
  3. Ввести промокод: ввести код и получить скидку на покупку подписок или покупку с магазина
  4. Ваши промокоды: просмотр истории и активные скидки

Реферальная система

Приглашение новых пользователей с бонусами.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Реферальная система"
  2. В меню "Реферальная система" выбрать "Ваша ссылка", "Ваши рефералы", "Топ рефералов"
  3. Ваша ссылка: скопировать реферальную ссылку и поделиться ей
  4. Ваши рефералы: посмотреть список рефералов и бонусы за них
  5. Топ рефералов: проверка и просмотр рейтинга и вашей позиции в топе рефералов

Рекламные каналы

Подписка на каналы с бонусом в виде дополнительного дня.

Инструкция по использованию:

  1. В меню "Подписка на бота" выбрать "Рекламные каналы"
  2. Нажать кнопку для подписки на канал, далее проверка подписки через кнопку, после успешного подтверждения начисляется +1 день к подписке

Калькуляторы

Расход топлива

Калькулятор "Расход топлива" предоставляет возможность рассчитать количество потраченного топлива и средств на поездку по заданному маршруту

Инструкция по использованию:

  1. В главном меню выберите "Калькуляторы"
  2. В меню "Калькуляторы" выберите "Расход топлива"
  3. Ввод маршрута:
    • Укажите начальное местоположение (через геолокацию или текст)
    • Укажите конечное местоположение
    • При необходимости добавьте дополнительные точки маршрута
  4. Расстояние:
    • Выберите "Автоматическое" для расчета расстояния по геолокации
    • Или выберите "Ввод вручную" и укажите расстояние в километрах
  5. Дата поездки:
    • Введите дату в формате или выберите "Пропустить ввод даты"
  6. Тип топлива:
    • Выберите тип топлива из списка: АИ-92, АИ-95, АИ-98, АИ-100, ДТ, ГАЗ
  7. Цена топлива:
    • Выберите "Актуальная цена" для средней цены по данным, если доступно
    • Или выберите "Ввести свою" и укажите цену за литр в рублях
  8. Расход топлива:
    • Укажите расход топлива на 100 км в литрах
  9. Количество пассажиров:
    • Введите количество пассажиров

Результат:

Бот выдаст детализированный отчет с указанием:

  • Местоположений маршрута
  • Даты поездки
  • Расстояния
  • Типа топлива и его цены
  • Расхода топлива
  • Количества пассажиров
  • Общего расхода топлива (в литрах)
  • Общей стоимости поездки
  • Стоимости на одного пассажира
  • Ссылки на маршрут в Яндекс.Картах

Просмотр поездок:

  • Выберите "Посмотреть поездки" в меню "Расход топлива"
  • Введите номера поездок для просмотра деталей
  • Или выберите "Поездки в EXCEL" для получения файла с данными

Удаление поездок:

  • Выберите "Удалить поездки" в меню "Расход топлива"
  • Введите номера поездок для удаления

Алкоголь

Калькулятор предназначен для оценки уровня алкоголя в крови (в промилле) на основе введенных пользователем данных. Он также предоставляет рекомендации по состоянию и времени вытрезвления.

Инструкция по использованию:

  1. В главном меню выберите "Калькуляторы"
  2. В меню "Калькуляторы" выберите "Алкоголь"
  3. Ввод данных:
    • Укажите пол (Мужской/Женский)
    • Введите вес в килограммах (например, 70)
    • Выберите напитки из списка (например, Пиво, Водка) и укажите их объем (в литрах или мл)
    • Укажите скорость употребления (например, «За 1 час»)
    • Укажите, как давно закончили пить (например, «Только что» или введите время в часах/минутах)

Основные функции

1. Как начать использовать бота

Чтобы начать использовать бота, отправьте команду /start. Если вы новый пользователь и не подписаны на Telegram-канал, необходимо подписаться на него и ознакомиться с информацией по использованию бота на сайте. После активации бота вам предоставляется 3 пробных дня использования. По истечении пробного периода необходимо приобрести подписку для дальнейшего использования бота.

2. Подписка на бота

2.1. Покупка подписки

В меню "Подписка на бота" выбрать "Купить подписку". Выбрать период через кнопки. Оплатить через Telegram Payments и Paymaster. Получить подтверждение оплаты и бонусы (баллы, скидки).

2.2. Отмена подписки

В меню "Подписка на бота" выберите "Возврат". В меню "Возврат" выберите "Отменить подписки". Просмотрите список активных подписок, доступных для возврата (указаны тип, оставшееся время, стоимость, возможный возврат). Введите номера подписок для отмены. Подтвердите отмену, выбрав "Подтвердить". Вы получите информацию о сумме возврата и комиссии. Ожидайте зачисления средств на ваш счет в течение 1–7 рабочих дней. Если возврат не выполнен, обратитесь в поддержку. Если даты других подписок изменились, вы получите уведомление с новым расписанием.

2.3. Просмотр активных и истёкших подписок

В меню "Подписка на бота" выберите "Ваши подписки". В меню "Ваши подписки" выбрать "Посмотреть подписки" или "История подписок".

2.4. Реферальная система

3. Описание функций

  • Количество пассажиров: укажите количество пассажиров в автомобиле.

После того как вы отправите все данные, бот произведет расчет и предоставит вам детализированный отчет о расходах на топливо для вашей поездки. Также у вас будет возможность сохранить поездку для дальнейшего просмотра или анализа.

Для просмотра сохраненных поездок выберите пункт «Просмотр поездок» в меню.

    • Модель: укажите модель транспорта.
    • Год выпуска: укажите год выпуска транспорта.

  • «За год» — для записей за конкретный год.
  • «За всё время» — для всех записей.

Чтобы удалить свои траты и ремонты по конкретному транспорту, выберите команду «Удалить траты» или «Удалить ремонты». Затем выберите:

  • «По категории» — для удаления по конкретной категории.
  • «За месяц» — для удаления записей за определенный месяц.
  • «За год» — для удаления записей за год.
  • «За всё время» — для удаления всех записей.

4. Система уведомлений

4.1. Уведомления о неактивности

Система отслеживает активность пользователя. В случае отсутствия активности в течение периода, превышающего или равного 3 дням, пользователю отправляется уведомление о неактивности. В уведомлении указывается, что данные пользователя будут удалены через 1 месяц при условии сохранения неактивности. Отсчет времени начинается с момента отправки первого уведомления. Если в течение указанного периода активность не возобновляется, данные пользователя удаляются без возможности восстановления.

4.2. Уведомления о погоде и ценах на топливо

Система предоставляет пользователю персонализированные уведомления о погоде и ценах на топливо. Уведомления формируются на основе данных, предоставленных пользователем:

  • Если пользователь указал только географические координаты (геопозицию), уведомление будет содержать исключительно информацию о погоде.
  • Если пользователь указал только город, уведомление будет содержать информацию о ценах на топливо.
  • Если пользователь предоставил и географические координаты, и город, уведомление будет содержать комбинированную информацию о погоде и ценах на топливо.

Данные уведомления отправляются автоматически на основе актуальных данных, полученных из внешних источников.
