Информация о боте

CAR MANAGER BOT (@CarMngrBot) — это мессенджерный многофункциональный Telegram бот для автомобилистов, который упрощает повседневные задачи, связанные с использованием и обслуживанием автомобиля.

Бот создан для повышения удобства планирования поездок, учета расходов и ремонтов, диагностики транспорта и получения актуальной информации о погоде, ценах на топливо, маршрутах и других важных аспектах.

Основные функции

Расчет топлива

Расчет количества топлива для поездки с учетом расстояния, расхода топлива и других параметров.

Инструкция по расчету топлива:

  1. Выберите пункт «Расчет топлива» в меню
  2. Укажите начальное и конечное местоположение
  3. Введите дату поездки в формате ДД.ММ.ГГГГ
  4. Укажите расстояние поездки в километрах
  5. Выберите тип топлива (АИ-92, ДТ, ГАЗ и др.)
  6. Введите цену топлива за литр
  7. Укажите расход топлива на 100 км
  8. Введите количество пассажиров
  9. Бот рассчитает и покажет детализированный отчет

Для просмотра сохраненных поездок выберите «Просмотр поездок», для удаления - «Удалить поездку».

Ваш транспорт

Управление данными о вашем транспорте (марка, модель, год выпуска, госномер).

Инструкция по управлению транспортом:

  1. Выберите пункт «Ваш транспорт» в меню
  2. Для добавления транспорта:
    • Введите марку транспорта
    • Укажите модель транспорта
    • Введите год выпуска
    • Добавьте госномер в формате РФ
  3. Для удаления транспорта выберите его из списка
  4. Для просмотра списка транспорта нажмите «Посмотреть транспорт»

Траты и ремонт

Учет расходов на транспорт и ремонты по категориям с возможностью анализа.

Инструкция по учету трат и ремонтов:

  1. Для записи новой траты используйте команду «Записать трату»
  2. Для записи нового ремонта используйте команду «Записать ремонт»
  3. Введите данные:
    • Выберите транспорт
    • Укажите категорию (можно добавить свою)
    • Введите название траты/ремонта
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите сумму
    • Добавьте описание (опционально)
  4. Для просмотра:
    • Выберите «Посмотреть траты» или «Посмотреть ремонты»
    • Можно фильтровать по категории, месяцу, году или за все время
  5. Для удаления:
    • Выберите «Удалить траты» или «Удалить ремонты»
    • Выберите критерий удаления (по категории, за месяц, за год или все)

Поиск транспорта

Сохранение местоположения транспорта и получение маршрута к нему.

Инструкция по поиску транспорта:

  1. Выберите команду «Найти транспорт»
  2. Укажите геопозицию транспорта (отправьте местоположение)
  3. Отправьте свою текущую геопозицию
  4. Бот предоставит ссылку на маршрут для поиска

Поиск мест

Поиск ближайших АЗС, моек, сервисов на основе вашей геопозиции.

Инструкция по поиску мест:

  1. Выберите команду «Поиск мест»
  2. Выберите категорию для поиска (АЗС, мойки и т.д.)
  3. Отправьте свою текущую геопозицию
  4. Бот предложит список ближайших мест с маршрутами

Погода

Прогноз погоды на сегодня, завтра, неделю или месяц для планирования поездок.

Инструкция по получению прогноза погоды:

  1. Выберите команду «Погода»
  2. Отправьте свою текущую геопозицию
  3. Выберите период прогноза:
    • На сегодня
    • На завтра
    • На неделю
    • На месяц
  4. Бот предоставит прогноз погоды на выбранный период

Цены на топливо

Актуальная информация о стоимости различных видов топлива в вашем городе.

Инструкция по получению цен на топливо:

  1. Выберите команду «Цены на топливо»
  2. Укажите название вашего города
  3. Выберите тип топлива:
    • АИ-92
    • АИ-95
    • АИ-98
    • АИ-100
    • ДТ
    • ГАЗ
  4. Бот предоставит актуальные цены на выбранное топливо

Код региона

Определение региона по госномеру или коду региона.

Инструкция по работе с кодами регионов:

  1. Выберите команду «Код региона»
  2. Для определения региона по номеру:
    • Введите госномер автомобиля в формате РФ
  3. Для поиска информации по коду региона:
    • Укажите номер региона
  4. Бот предоставит информацию о транспорте или регионе

Анти-радар

Отображение камер видеофиксации нарушений ПДД.

Инструкция по использованию анти-радара:

  1. Выберите команду «Анти-радар»
  2. Отправьте свою текущую геопозицию
  3. Бот покажет ближайшие камеры на основе вашего местоположения
  4. Для обновления данных отправьте новую геопозицию

Примечание: Обновление геопозиции происходит вручную.

Напоминания

Создание напоминаний о важных задачах или событиях.

Инструкция по работе с напоминаниями:

  1. Выберите команду «Напоминания»
  2. Введите название напоминания
  3. Укажите дату напоминания
  4. Введите время напоминания
  5. Бот отправит уведомление в установленное время
  6. Для просмотра напоминаний выберите соответствующий пункт в меню
  7. Для удаления напоминания выберите его из списка

Коды ошибок OBD2

Диагностика автомобиля через поиск и расшифровку кодов ошибок OBD2.

Инструкция по диагностике OBD2:

  1. Выберите команду «Коды OBD2»
  2. Введите код ошибки, полученный через диагностический прибор
  3. Бот предоставит расшифровку введенного кода

Новости

Просмотр актуальных новостей в режиме реального времени.

Инструкция по работе с новостями:

  1. Выберите соответствующий интерактивный элемент интерфейса
  2. Просматривайте актуальные новости
  3. Новости обновляются администратором в режиме реального времени

Рекламные заявки

Подача заявки на размещение рекламы с уведомлением администратора.

Инструкция по подаче рекламных заявок:

  1. Выберите соответствующий пункт в меню
  2. Заполните форму заявки на размещение рекламы
  3. После отправки администратор получит уведомление
  4. Вы можете просматривать список своих заявок
  5. При необходимости можно отозвать заявку

Чат с администратором

Инициация чата с администратором для решения критических вопросов.

Инструкция по чату с администратором:

  1. Выберите соответствующий пункт в меню
  2. Вы можете инициировать чат с администратором до 3 раз в 24 часа
  3. Функция предназначена для решения критических вопросов
  4. Несоответствующие заявки будут отклонены
  5. Администратор может связаться с вами напрямую
  6. Вы можете принять или отклонить запрос на чат

Система уведомлений

Уведомления о неактивности

Уведомление при неактивности более 3 дней, удаление данных через месяц.

Информация о уведомлениях:

  • Система отслеживает вашу активность
  • При неактивности более 3 дней вы получите уведомление
  • Данные будут удалены через 1 месяц, если активность не возобновится
  • Отсчет начинается с момента первого уведомления
  • После удаления данные восстановить невозможно

Погода и топливо

Персонализированные уведомления о погоде и ценах на топливо.

Информация о уведомлениях:

  • Уведомления формируются на основе ваших данных
  • Если указана только геопозиция - получаете уведомления о погоде
  • Если указан только город - получаете цены на топливо
  • Если указаны оба параметра - получаете комбинированные уведомления
  • Данные берутся из внешних источников в автоматическом режиме

Инструкция по использованию бота

1. Как начать использовать бота

Чтобы начать использовать бота, отправьте команду /start. Если вы новый пользователь и не подписаны на Telegram-канал, необходимо подписаться на него и ознакомиться с информацией по использованию бота на сайте. После активации бота вам предоставляется 3 пробных дня использования. По истечении пробного периода необходимо приобрести подписку для дальнейшего использования бота.

2. Подписка на бота

2.1. Покупка подписки

Чтобы приобрести подписку, выберите период (неделя, месяц, год). При покупке вы автоматически соглашаетесь с условиями использования бота. Администрация не несет ответственности за потраченные вами средства.

После оплаты активируется оплаченный период. Подписка действует следующим образом: сначала заканчивается пробный период, а затем начинается оплаченный. Дни всех подписок суммируются.

2.2. Отмена подписки

Чтобы отменить подписку, выберите её из списка (*исключения: пробный период, бесплатный день за приглашенного реферала, а также подписка, назначенная администратором, если таковая имеется).

После выбора подписки вы получите сообщение о возврате средств. Возврат осуществляется за вычетом неиспользованных дней. Если подписка была куплена сегодня, она считается использованной (= 1 день), и средства за неё не возвращаются.

2.3. Просмотр активных и истёкших подписок

Вы можете посмотреть свои активные подписки, нажав кнопку «Посмотреть подписку». В разделе «История подписок» хранятся данные о ваших истёкших подписках.

2.4. Реферальная система

У вас есть реферальная ссылка, с помощью которой вы можете приглашать друзей и знакомых. За каждого приглашённого пользователя вы получаете +1 бесплатный день использования бота. Также вы можете просмотреть список приглашённых вами людей.

3. Описание функций

3.1. Расчет топлива и сохранение поездок

Для расчета расхода топлива выберите пункт «Расчет топлива» в меню. Затем отправьте следующие данные:

  • Начальное местоположение: укажите ваше начальное местоположение.
  • Конечное местоположение: укажите конечное местоположение.
  • Дата поездки: укажите дату поездки в формате ДД.ММ.ГГГГ.
  • Расстояние: укажите расстояние поездки в километрах.
  • Тип топлива: укажите тип топлива (например, АИ-92, ДТ, ГАЗ).
  • Цена топлива за литр: укажите цену за литр топлива в рублях.
  • Расход топлива на 100 км: укажите расход топлива вашего автомобиля на 100 км.
  • Количество пассажиров: укажите количество пассажиров в автомобиле.

После того как вы отправите все данные, бот произведет расчет и предоставит вам детализированный отчет о расходах на топливо для вашей поездки. Также у вас будет возможность сохранить поездку для дальнейшего просмотра или анализа.

Для просмотра сохраненных поездок выберите пункт «Просмотр поездок» в меню.

Если вы хотите удалить сохраненную поездку, выберите пункт «Удалить поездку» и следуйте инструкциям бота.

3.2. Ваш транспорт

Чтобы управлять транспортом, выберите пункт «Ваш транспорт» в меню. В этом разделе доступны следующие функции:

  • Добавить транспорт: Введите данные вашего транспорта:
    • Марка: укажите марку транспорта.
    • Модель: укажите модель транспорта.
    • Год выпуска: укажите год выпуска транспорта.
    • Госномер: введите государственный регистрационный номер в формате РФ.
  • Удалить транспорт: Выберите транспорт, который вы хотите удалить, из списка сохраненных.
  • Посмотреть транспорт: Получите список всех добавленных транспортных средств с их основными характеристиками.

После добавления транспорта он станет доступным для использования в других функциях бота, таких как траты и ремонты.

3.3. Траты и ремонты

Чтобы записать трату или ремонт, выберите соответствующую команду в меню:

  • Для записи новой траты используйте команду «Записать трату».
  • Для записи нового ремонта используйте команду «Записать ремонт».

Введите следующие данные:

  • Транспорт — выберите транспорт для записи.
  • Категория — выберите категорию для записи (вы можете добавить свою категорию или удалить ее).
  • Название траты или ремонта.
  • Дата в формате ДД.ММ.ГГГГ.
  • Сумма траты или ремонта.
  • Описание для траты или ремонта.

Чтобы просмотреть свои траты и ремонты по конкретному транспорту, выберите команду «Посмотреть траты» или «Посмотреть ремонты». Вы можете выбрать фильтрацию по:

  • «По категории» — для просмотра по конкретной категории.
  • «За месяц» — для просмотра записей за конкретный месяц.
  • «За год» — для записей за конкретный год.
  • «За всё время» — для всех записей.

Чтобы удалить свои траты и ремонты по конкретному транспорту, выберите команду «Удалить траты» или «Удалить ремонты». Затем выберите:

  • «По категории» — для удаления по конкретной категории.
  • «За месяц» — для удаления записей за определенный месяц.
  • «За год» — для удаления записей за год.
  • «За всё время» — для удаления всех записей.

3.4. Поиск транспорта

Для поиска транспорта выберите команду «Найти транспорт» и укажите следующие данные:

  • Геопозиция транспорта: отправьте местоположение транспорта.
  • Ваша геопозиция: отправьте свое текущее местоположение.

Бот предоставит вам ссылку на маршрут для поиска.

3.5. Поиск мест

Для поиска ближайших мест выберите команду «Поиск мест» и выполните следующие действия:

  • Категория: выберите категориу для поиска.
  • Геопозиция: отправьте свою текущую геопозицию.

Бот предложит вам список ближайших мест и выдаст ссылки с маршрутом на основе вашей локации.

3.6. Погода

Чтобы узнать прогноз погоды, выберите команду «Погода» и выполните следующие действия:

  • Геопозиция: отправьте свою текущую геопозицию.
  • Период прогноза: выберите один из доступных вариантов:
    • На сегодня.
    • На завтра.
    • На неделю.
    • На месяц.

Бот предоставит прогноз погоды на выбранный период.

3.7. Цены на топливо

Для получения актуальных цен на топливо выберите команду «Цены на топливо» и выполните следующие действия:

  • Город: укажите название вашего города.
  • Тип топлива: выберите один из доступных вариантов:
    • АИ-92.
    • АИ-95.
    • АИ-98.
    • АИ-100.
    • ДТ.
    • ГАЗ.

Бот предоставит актуальные цены на выбранное топливо в вашем городе.

3.8. Код региона

Для получения информации о регионе по госномеру транспортного средства выберите команду «Код региона» и укажите следующие данные:

  • Государственный номер: введите номер автомобиля в формате РФ.
  • Регион: укажите номер региона.

Бот предоставит информацию о транспорте или регионе на основе введенных данных.

3.9. Анти-радар

Для получения информации о камерах видеонаблюдения, фиксирующих нарушения ПДД, выберите команду «Анти-радар» и выполните следующие действия:

  • Геопозиция: отправьте свою текущую геопозицию.

Обновление геопозиции происходит вручную. Бот покажет ближайшие камеры на основе вашей геопозиции.

3.10. Напоминания

Чтобы установить напоминание, выберите команду «Напоминания» и укажите следующие данные:

  • Если пользователь предоставил и географические координаты, и город, уведомление будет содержать комбинированную информацию о погоде и ценах на топливо.
