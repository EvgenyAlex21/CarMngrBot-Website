Информация о боте

CAR MANAGER BOT (@CarMngrBot) — это мессенджерный многофункциональный Telegram бот для автомобилистов, который упрощает повседневные задачи, связанные с использованием и обслуживанием автомобиля.

Бот создан для повышения удобства планирования поездок, учета расходов и ремонтов, диагностики транспорта и получения актуальной информации о погоде, ценах на топливо, маршрутах и других важных аспектах.

Подписка на бота

Управление подписками и доступом к функциям бота. Поддержка бесплатных и платных подписок, пробных периодов и реферальной системы.

    • Если переход был по реферальной ссылке, то +1 день. Итого 4 дня для старта

  3. Приобрести подписку через Telegram Payments
  4. Просмотр статусов подписок и истории
  5. Управление баллами и промокодами
  6. Участие в реферальной программе
  7. Получение бонусов за подписку на рекламные каналы

Калькуляторы

Доступ к различным автомобильным калькуляторам для расчетов и планирования.

Инструкция по использованию калькуляторов:

  1. В главном меню выберите "Калькуляторы"
  2. Выберите нужный калькулятор из списка:
    • Расход топлива
    • Алкоголь
    • Автокредит
    • Налог
    • Растаможка
    • ОСАГО
    • Шины
  3. Для каждого калькулятора следуйте инструкциям в соответствующем разделе

Траты и ремонты

Учет расходов на транспорт и ремонты по категориям с возможностью анализа.

Инструкция по учету трат и ремонтов:

  1. Перед тем, как делать записи, необходимо выбрать или добавить транспорт из меню «Ваш транспорт»
    • Выберите «Добавить транспорт» для создания новой записи
    • Введите данные:
      • Марка
      • Модель
      • Год
      • Госномер
    • Вы можете воспользоваться также следующими функциями:
      • «Посмотреть транспорт» для просмотра списка ваших транспортных средств
      • «Удалить транспорт» - для удаления ненужной записи
      • «Изменить транспорт» - для изменения полей траснпорта
  2. Для записи новой траты/ремонта используйте команду «Записать трату/Записать ремонт»
  3. Введите данные:
    • Выберите транспорт
    • Укажите категорию
    • Введите название траты/ремонта
    • Добавьте описание (можно пропустить)
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите сумму
  4. Для просмотра:
    • Выберите «Посмотреть траты/Посмотреть ремонты»
    • Можно фильтровать по:
      • Транспорту
      • Категории
      • Месяцу
      • Году
      • За все время
  5. Для удаления:
    • Выберите «Удалить траты/Удалить ремонты»
    • Выберите критерий удаления:
      • По транспорту
      • По категории
      • За месяц
      • За год
      • За все время

Найти транспорт

Помощь в поиске маршрута от текущей геопозиции до местоположения транспорта.

Инструкция по использованию:

  1. В главном меню выберите "Найти транспорт"
  2. Отправьте геопозицию транспорта
  3. Отправьте свою геопозицию
  4. Получите ссылку с маршрутом в Яндекс.Картах
  5. Если процесс был прерван, выберите "Продолжить" или "Начать заново"

Поиск мест

Поиск ближайших мест по вашей геопозиции.

Инструкция по использованию:

  1. В главном меню выберите "Поиск мест"
  2. Выберите категорию поиска:
    • АЗС
    • Автомойки
    • Автосервисы
    • Парковки
    • Эвакуация
    • ГИБДД
    • Комиссары
    • Штрафстоянка
  3. Отправьте свою геопозицию
  4. Получите адрес и ссылку на Яндекс.Карты с ближайшими местами по выбранной категории

Погода

Получение прогноза погоды по геопозиции или названию города.

Инструкция по использованию:

  1. В главном меню выберите "Погода"
  2. Отправьте геопозицию или введите название города
  3. Выберите период:
    • Сегодня
    • Завтра
    • Неделя
    • Месяц
  4. Получите данные о погоде за выбранный период

Цены на топливо

Узнайте средние цены на топливо в выбранном городе с возможностью просмотра по геопозиции или названию города.

Инструкция по использованию:

  1. В главном меню выберите "Цены на топливо"
  2. Введите город, выберите из последних или отправьте геопозицию
  3. Выберите тип топлива:
    • АИ-92
    • АИ-95
    • АИ-98
    • АИ-100
    • ДТ
    • Газ
  4. Получите список средних цен по брендам АЗС (стандарт и премиум)

Код региона

Определение региона РФ по коду региона или государственному номеру автомобиля.

Инструкция по использованию:

  1. В главном меню выберите "Код региона"
  2. Введите код региона РФ или госномер автомобиля РФ
  3. Получите информацию:
    • Название региона
    • Для госномеров: ссылка на AvtoCod с проверкой авто

Коды OBD2

Расшифровка кодов ошибок OBD2 по диагностическому прибору.

Инструкция по использованию:

  1. В главном меню выберите "Коды OBD2"
  2. Введите код ошибки, полученный через диагностический прибор
  3. Бот предоставит краткую расшифровку введенного кода

Напоминания

Создание напоминаний с различными типами повторений.

Инструкция по использованию:

  1. В главном меню выберите "Напоминания"
  2. Добавление напоминаний:
    • Введите название
    • Выберите тип повторения
      • Однократное
      • Ежедневное
      • Еженедельное
      • Ежемесячное
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите время в формате ЧЧ:ММ
  3. Просмотр напоминаний:
    • Выберите статус:
      • Активные
      • Истекшие
    • Выберите тип напоминания для показа
  4. Удаление напоминаний:
    • Выберите статус:
      • Активные
      • Истекшие
    • Выберите тип напоминания
    • Укажите номера напоминаний из списка для удаления

Антирадар

Отслеживание дорожных камер и уведомление о приближении к ним на основе геопозиции.

Инструкция по использованию:

  1. В главном меню выберите "Антирадар"
  2. Отправьте геопозицию для активации антирадара (обновление геопозиции происходит вручную из-за ограничений на стороне Telegram)
  3. Получайте уведомления о камерах
  4. Для выключения антирадара используйте кнопку "Выключить антирадар"

Прочее

Объединяет функции: новости, курсы валют, уведомления, реклама и чат с администратором.

Инструкция по использованию:

  1. В главном меню выберите "Прочее"
  2. В меню "Прочее" выберите нужную функцию:
    • Новости: выберите количество новостей (3–15) для просмотра
    • Курсы валют: просмотрите актуальные курсы валют
    • Уведомления: включите или выключите уведомления на погоду, цены на топливо, курсы или на все сразу
    • Реклама:
      • "Заявка на рекламу": введите тему, даты, время, текст, медиа
      • "Ваши заявки": просмотрите или отмените заявки
    • Чат с админом:
      • Ввод темы чата
      • Ожидание принятия или отклонения вашего запроса администратором
      • В активном чате вы можете отправлять текст или мультимедийные файлы
      • Завершение чата происходит через "Стоп" или /stopchat, либо автоматическое завершение по истечению 3 минут

Отмена подписок

Отмена активных подписок с возвратом средств.

Инструкция по использованию:

  1. В меню "Подписка на бота" выберите "Возврат".
  2. В меню "Возврат" выберите "Отменить подписки".
  3. Просмотрите список активных подписок, доступных для возврата:
    • Тип
    • Оставшееся время
    • Стоимость
    • Возможный возврат
  4. Введите номера подписок для отмены.
  5. Подтвердите отмену, выбрав "Подтвердить". Вы получите информацию о сумме возврата и комиссии.
  6. Ожидайте зачисления средств на ваш счет в течение 1–7 рабочих дней. Если возврат не выполнен, обратитесь в поддержку.
  7. Если даты других подписок изменились, вы получите уведомление с новыми датами.

Просмотр истории возвратов с подробной информацией.

  1. В меню "Подписка на бота" выберите "Возврат"
  2. В меню "Возврат" выберите "История возвратов"
  3. Просмотрите список всех возвратов с подробной информацией:
    • Сумма
    • Дата
    • Статус
    • Ошибки
  4. При изменении статуса возврата вы получите уведомление:
    • Успешный возврат

Расход топлива

Калькулятор предназначен для рассчета количества потраченного топлива и средств на поездку по заданному маршруту.

Инструкция по использованию:

Инструкция

1. Как начать использовать бота

Для начала работы с ботом выполните следующие шаги:

  1. Отправьте команду /start.
  2. Подпишитесь на Telegram-канал, указанный в боте.
  3. Ознакомьтесь с информацией на сайте.
  4. Получите 3-дневный пробный период после активации.
  5. По окончании пробного периода оформите подписку для продолжения использования.
  6. Пригласите друга по реферальной ссылке, чтобы получить дополнительный день пробного периода.

2. Подписка на бота

2.1. Покупка подписки

Для покупки подписки выполните следующие действия:

  1. В меню "Подписка на бота" выберите "Купить подписку".
  2. Укажите период подписки:
    • 3 дня
    • 7 дней
    • 30 дней
    • 90 дней
    • 180 дней
    • 365 дней
  3. Примените скидки (пользовательские или акционные), если доступны.
  4. Оплатите через Telegram Payments или Paymaster.
  5. Получите подтверждение оплаты и бонусы:
    • За первую подписку: от 3 до 15 баллов (зависит от периода).
    • За повторную подписку: от 0.5 до 10 баллов.
    • Специальное предложение: 7 дней за 99 ₽ (доступно через 15 дней после окончания подписки).

3.2. Алкоголь

Для расчета уровня алкоголя в крови (промилле) по формуле Видмарка:

  1. Параметры для расчета алкоголя::
    • Пол:
      • Мужской
      • Женский
    • Вес (кг.)
    • Тип напитков:
      • Кефир, Коньяк, Ликер десертный, Ликер крепкий, Ликер-крем, Лимонад, Лимончелло
      • Медовуха, Наливка, Настойка, Абсент, Бренди, Вермут, Вино игристое
      • Вино крепленое, Вино некрепкое, Виски, Водка, Джин, Квас, Пиво
      • Пиво безалкогольное, Пиво крепкое, Самбука, Ром, Сидр, Текила, Тоник
    • Объем напитков:
      • л.
      • мл.
    • Крепость напитков (%)
    • Скорость употребления:
      • Сразу
      • За 1-10 часов
    • Время с последнего приема:
      • Только что
      • Ввод своего времени в часах или минутах
    • Наличие еды:
      • Нет
      • Закуски
      • Пища
  2. Учитывает следущие параметры:
    • Пол:
      • Мужчины: 0.7
      • Женщины: 0.6
    • Скорость:
      • Сразу: -0%
      • За 1-10 часов: -n%
    • Еда:
      • Нет: -0%
      • Закуски: -10%
      • Пища: -30%
    • Выведение:
      • Мужчины: 0.15 промилле/час
      • Женщины: 0.10 промилле/час
  3. Выводит результат:
    • Текущий уровень алкоголя (%)
    • Время до полного вытрезвления
    • Дата и время вытрезвления
    • Рекомендации:
      • 0.0-0.3 промилле: слабое опьянение
      • 0.3-0.6 промилле: среднее опьянение
      • 0.6-1.0 промилле: сильное опьянение
      • 1.0-2.0 промилле: тяжелое опьянение
      • 2.0-3.0 промилле: алкогольное отравление
      • 3.0+ промилле: опасно для жизни
  4. Просмотр и удаление:
    • Выберите «Посмотреть алкоголь» и введите номера расчетов
    • Возможен экспорт в Excel
    • Выберите «Удалить алкоголь» и введите номера расчетов для удаления

Примечание:

Формулы расчета:

  • C = A / W * r, где:
    • C - концентрация алкоголя в крови (промилле)
    • A - масса чистого алкоголя в граммах (V * 1000 * S * 0.79)
    • W - вес человека в кг.
    • r - коэффициент распределения (0.7 для мужчин, 0.6 для женщин)
    • V - объем напитка в литрах
    • S - крепость напитка в долях
    • p (этилового спирта) - 0.79

Важная информация:

  • Калькулятор не сможет дать 100% точный результат! Если вы выпили, то НИ в коем случае нельзя садиться за руль после алкоголя как минимум в течение суток!!!
  • Уровень алкоголя в крови зависит от множества индивидуальных факторов (метаболизм, состояние здоровья, прием лекарств)
  • Снижение концентрации за счет еды и скорости употребления является приблизительным
  • Даже малые дозы алкоголя могут влиять на скорость реакции и внимание
  • В РФ допустимая норма алкоголя за рулем составляет 0.3 промилле (погрешность измерения)

      • Аннуитет

  3. Выводит результат:

      • Полная

  4. Просмотр и удаление:

    • Выберите «Удалить автокредит» и введите номера расчетов для удаления

Примечание:

Формулы расчета:

  • Аннуитетный платеж: P = (Principal * r * (1+r)^n)/((1+r)^n - 1), где:
    • P - ежемесячный платеж
    • Principal - сумма кредита
    • r - месячная процентная ставка
    • n - количество месяцев
  • Дифференцированные платежи:
    • Основной долг делится на равные части
    • Проценты начисляются на остаток

Важная информация:

  • Калькулятор выдает общую информацию для получения автокредита. Для реальных деталей стоит обратиться в банк или рассчитать на их сайте

Формулы расчета:

    • Повышающий коэффициент для ТС >10 млн руб.:
      • 3.0 (до 3 лет)
      • 2.0 (3–5 лет)
      • 1.5 (5–10 лет)
      • 1.0 (все остальные случаи)
    • Льгота - коэффициент льготы:
      • 1.0 (без льгот)
      • 0.0 (с льготами)

Важная информация:

  • Калькулятор выдает информацию согласно актуальным ставкам по выбранному году налогооблажение. Для автомобилей со стоимостью >10 миллионов рублей используются данные на 2025 год
  • К льготным катеогиям относятся:

    • Участники специальной военной операции
    • Инвалиды и лица, подвергшиеся радиации
    • Родители ребенка-инвалида до 18 лет

Примечание:

Формулы расчета:

  • Таможенный сбор = фиксированная ставка в зависимости от стоимости авто
  • Таможенная пошлина = ставка * (объем двигателя или стоимость)
  • Утилизационный сбор = базовая ставка * коэффициент (зависит от возраста и объема)
  • Акциз = ставка (зависит от мощности) * мощность л.с.
  • НДС = 0.2 * (стоимость + пошлина + акциз)

Важная информация:

  • Калькулятор выдает ориентировочную информацию для растаможки автомобиля по РФ. Более точные данные уточняйте из других источников
  • Для актуальных курсов валют используется сервис ЦБ РФ

  • Текстовые сообщения (например, ввод марки автомобиля).
  • Интерактивные кнопки (например, выбор типа подписки).
  • Геолокацию (отправка координат через Telegram).
  • Команды Бота (например, /calc_fuel для калькулятора топлива).

Дополнительные данные могут поступать от сторонних сервисов при выполнении запросов Пользователя:

  • Яндекс.Карты: Данные о маршрутах, местоположении АЗС, автомоек.
  • OpenWeatherMap: Прогноз погоды по координатам.
  • AvtoCod: Информация о транспортных средствах (например, по VIN).
  • Telegram Payments: Подтверждение транзакций.

Данные от сторонних сервисов запрашиваются только с согласия Пользователя (например, при отправке геолокации).

2. Обработка данных

2.1. Методы обработки

Обработка данных включает:

  • Сбор: Получение данных через Telegram API (например, сообщение с координатами).
  • Хранение: Сохранение в зашифрованных базах данных на серверах в Российской Федерации.
  • Анализ: Обработка для выполнения функций (например, вычисление расхода топлива).
  • Передача: Отправка необходимых данных сторонним сервисам (например, координаты в Яндекс.Карты).
  • Удаление: Уничтожение данных по запросу или при истечении срока хранения.

Обработка осуществляется автоматизированно с использованием серверов, соответствующих требованиям законодательства РФ.

2.2. Принципы обработки

Разработке данных осуществляется на основе следующих принципов:

  • Законность: Обработка только с согласия Пользователя или в рамках законодательства.
  • Минимизация: Сбор минимально необходимого объема данных.
  • Конфиденциальность: Ограничение доступа к данным для неавторизованных лиц.
  • Точность: Обеспечение актуальности и достоверности данных.

Пример: Если Пользователь отправляет геолокацию для поиска АЗС, координаты обрабатываются только для этого запроса и не сохраняются после выполнения.

3. Защита данных

3.1. Технические и организационные меры

Для защиты данных применяются следующие меры:

  • Шифрование: Хранение данных с использованием AES-256, передача через HTTPS.
  • Контроль доступа: Доступ к данным только у уполномоченных сотрудников с двухфакторной аутентификацией.
  • Резервное копирование: Ежедневное создание зашифрованных копий данных с хранением в течение 7 дней.
  • Аудит безопасности: Ежеквартальный анализ уязвимостей и тестирование на проникновение.
  • Мониторинг: Постоянное наблюдение за серверной активностью для выявления угроз.
  • Обучение персонала: Регулярное обучение сотрудников по вопросам защиты данных.

Пример: Данные о тратах хранятся в зашифрованном виде, и даже в случае компрометации сервера они не могут быть прочитаны без ключа.

3.2. Возможные риски

Несмотря на принятые меры, ни одна система не является полностью защищенной. Возможные риски включают:

  • Несанкционированный доступ из-за компрометации устройства Пользователя.
  • Кибератаки (например, фишинг, DDoS).
  • Ошибки Пользователя (например, передача Telegram ID третьим лицам).

Пользователь обязан обеспечивать безопасность своего устройства, использовать надежные пароли и не делиться учетными данными.

4. Передача данных

Данные Пользователя передаются третьим лицам только в следующих случаях:

  • Сторонние сервисы: Передача минимально необходимых данных для выполнения функций Бота. Полный список:
    • Яндекс.Карты: Координаты для построения маршрутов и поиска мест.
    • OpenWeatherMap: Координаты для прогноза погоды.
    • Автокод: VIN-код или госномер для проверки автомобиля.
    • Telegram Payments/Paymaster: Платежные данные для обработки транзакций.
  • Законные запросы: Передача данных по решению суда, запросу правоохранительных органов или в иных случаях, предусмотренных законом.

Пример: При запросе маршрута до АЗС координаты отправляются в Яндекс.Карты, но не хранятся там после выполнения запроса.

Разработчик обеспечивает, что передаваемые данные минимальны и защищены соглашениями о конфиденциальности со сторонними сервисами.

5. Хранение данных

5.1. Сроки хранения

Данные хранятся в течение всего периода использования Бота. Сроки хранения для отдельных категорий:

  • Идентификационные данные: До удаления аккаунта или запроса на удаление.
  • Финансовые данные: До 3 лет (в соответствии с с налоговым законодательством).
  • Геолокационные данные: До выполнения запроса (например, поиск АЗС), затем удаляются.
  • Данные напоминаний: До выполнения задачи или удаления Пользователем.

При неактивности Пользователя более 60 дней данные удаляются на удаление с уведомлением за 30 дней через Telegram.

5.2. Удаление данных

Пользователь может запросить удаление данных через команду /delete_data или обращение в чат с администратором. Процесс удаления:

  • Регистрация запроса.
    • Пользователь указывает Telegram ID и подтверждает личность.
  • Обработка: Удаление данных из активных баз в течение 48 часов.
  • Резервные копии: Удаление из резервов в течение 7 рабочих дней.
  • Подтверждение: Уведомление Пользователя об удалении.

Пример: Пользователь отправляет /delete_data, получает запрос на подтверждение, данные удаляются, и он получает сообщение: «Ваши данные успешно удалены».
