Калькуляторы

Доступ к различным автомобильным калькуляторам для расчетов и планирования.

Инструкция по использованию калькуляторов:

  1. В главном меню выберите "Калькуляторы"
  2. Выберите нужный калькулятор из списка:
    • ⭐ Расход топлива
    • ✓ Алкоголь
    • ⭐ Автокредит
    • ✓ Налог
    • ⭐ Растаможка
    • ⭐ ОСАГО
    • ⭐ Шины
  3. Для каждого калькулятора следуйте инструкциям в соответствующем разделе

✓ - доступно бесплатно

⭐ - требуется премиум подписка

ПРЕМИУМ

Траты и ремонты

Учет расходов на транспорт и ремонты по категориям с возможностью анализа.

Инструкция по учету трат и ремонтов:

  1. Перед тем, как делать записи, необходимо выбрать или добавить транспорт из меню «Ваш транспорт»
    • Выберите «Добавить транспорт» для создания новой записи
    • Введите данные:
      • Марка
      • Модель
      • Год
      • Госномер
    • Вы можете воспользоваться также следующими функциями:
      • «Посмотреть транспорт» для просмотра списка ваших транспортных средств
      • «Удалить транспорт» - для удаления ненужной записи
      • «Изменить транспорт» - для изменения полей траснпорта
  2. Для записи новой траты/ремонта используйте команду «Записать трату/Записать ремонт»
  3. Введите данные:
    • Выберите транспорт
    • Укажите категорию
    • Введите название траты/ремонта
    • Добавьте описание (можно пропустить)
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите сумму
  4. Для просмотра:
    • Выберите «Посмотреть траты/Посмотреть ремонты»
    • Можно фильтровать по:
      • Транспорту
      • Категории
      • Месяцу
      • Году
      • За все время
  5. Для удаления:
    • Выберите «Удалить траты/Удалить ремонты»
    • Выберите критерий удаления:
      • По транспорту
      • По категории
      • За месяц
      • За год
      • За все время
БЕСПЛАТНО

Найти транспорт

Помощь в поиске маршрута от текущей геопозиции до местоположения транспорта.

Инструкция по использованию:

  1. В главном меню выберите "Найти транспорт"
  2. Отправьте геопозицию транспорта
  3. Отправьте свою геопозицию
  4. Получите ссылку с маршрутом в Яндекс.Картах
  5. Если процесс был прерван, выберите "Продолжить" или "Начать заново"
ПРЕМИУМ

Поиск мест

Поиск ближайших мест по вашей геопозиции.

Инструкция по использованию:

  1. В главном меню выберите "Поиск мест"
  2. Выберите категорию поиска:
    • АЗС
    • Автомойки
    • Автосервисы
    • Парковки
    • Эвакуация
    • ГИБДД
    • Комиссары
    • Штрафстоянка
  3. Отправьте свою геопозицию
  4. Получите адрес и ссылку на Яндекс.Карты с ближайшими местами по выбранной категории
ПРЕМИУМ

Погода

Получение прогноза погоды по геопозиции или названию города.

Инструкция по использованию:

  1. В главном меню выберите "Погода"
  2. Отправьте геопозицию или введите название города
  3. Выберите период:
    • Сегодня
    • Завтра
    • Неделя
    • Месяц
  4. Получите данные о погоде за выбранный период
ПРЕМИУМ

Цены на топливо

Узнайте средние цены на топливо в выбранном городе с возможностью просмотра по геопозиции или названию города.

Инструкция по использованию:

  1. В главном меню выберите "Цены на топливо"
  2. Введите город, выберите из последних или отправьте геопозицию
  3. Выберите тип топлива:
    • АИ-92
    • АИ-95
    • АИ-98
    • АИ-100
    • ДТ
    • Газ
  4. Получите список средних цен по брендам АЗС (стандарт и премиум)
БЕСПЛАТНО

Код региона

Определение региона РФ по коду региона или государственному номеру автомобиля.

Инструкция по использованию:

  1. В главном меню выберите "Код региона"
  2. Введите код региона РФ или госномер автомобиля РФ
  3. Получите информацию:
    • Название региона
    • Для госномеров: ссылка на AvtoCod с проверкой авто
БЕСПЛАТНО

Коды OBD2

Расшифровка кодов ошибок OBD2 по диагностическому прибору.

Инструкция по использованию:

  1. В главном меню выберите "Коды OBD2"
  2. Введите код ошибки, полученный через диагностический прибор
  3. Бот предоставит краткую расшифровку введенного кода
БЕСПЛАТНО

Напоминания

Создание напоминаний с различными типами повторений.

Инструкция по использованию:

  1. В главном меню выберите "Напоминания"
  2. Добавление напоминаний:
    • Введите название
    • Выберите тип повторения
      • Однократное
      • Ежедневное
      • Еженедельное
      • Ежемесячное
    • Укажите дату в формате ДД.ММ.ГГГГ
    • Введите время в формате ЧЧ:ММ
  3. Просмотр напоминаний:
    • Выберите статус:
      • Активные
      • Истекшие
    • Выберите тип напоминания для показа
  4. Удаление напоминаний:
    • Выберите статус:
      • Активные
      • Истекшие
    • Выберите тип напоминания
    • Укажите номера напоминаний из списка для удаления
БЕСПЛАТНО

Антирадар

Отслеживание дорожных камер и уведомление о приближении к ним на основе геопозиции.

Инструкция по использованию:

  1. В главном меню выберите "Антирадар"
  2. Отправьте геопозицию для активации антирадара (обновление геопозиции происходит вручную из-за ограничений на стороне Telegram)
  3. Получайте уведомления о камерах
  4. Для выключения антирадара используйте кнопку "Выключить антирадар"
БЕСПЛАТНО ПРЕМИУМ

Прочее

Объединяет функции: новости, курсы валют, уведомления, бланки, реклама и чат с администратором.

Инструкция по использованию:

  1. В главном меню выберите "Прочее"
  2. В меню "Прочее" выберите нужную функцию:
    • ✓ Новости: выберите количество новостей (3–15) для просмотра
    • ✓ Курсы валют: просмотрите актуальные курсы валют
    • ⭐ Уведомления: включите или выключите уведомления на погоду, цены на топливо, курсы или на все сразу
    • ✓ Бланки: получите пусте бланки для заполнения: договора купли-продажи, европротокола, доверенности
    • ✓ Реклама:
      • "Заявка на рекламу": введите тему, даты, время, текст, медиа
      • "Ваши заявки": просмотрите или отмените заявки
    • ✓ Чат с админом:
      • Ввод темы чата
      • Ожидание принятия или отклонения вашего запроса администратором
      • В активном чате вы можете отправлять текст или мультимедийные файлы
      • Завершение чата происходит через "Стоп" или /stopchat, либо автоматическое завершение по истечению 3 минут

СИСТЕМНОЕ

Вспомогательные функции

Автоматические системы бота для обеспечения безопасности, контроля доступа и стабильности работы.

Системные компоненты:

  1. Внутренние системы обеспечивают работу бота:
    • Состояние функций: проверяет доступность функций, блокирует недоступные с информативным сообщением
    • Статистика функций: анализирует использование функций в обезличенной форме
    • Блокировка пользователей: проверяет статус пользователя, ограничивает доступ заблокированным
    • Обновление активности: фиксирует действия и дату последней активности
    • Состояние чата: предотвращает отправку повторных сообщений при обработке первого
  2. Системы безопасности и контроля:
    • Логирование ошибок: фиксирует технические проблемы без записи личных данных
    • Блокировка бота: определяет статус блокировки, сохраняет данные до 30 дней
    • Проверка функций: контролирует доступ к платным функциям по наличию подписки
    • Проверка подписки: требует активную подписку на основной канал бота
  3. Взаимодействие с пользователем:
    • Текстовые сообщения: обрабатывает разные типы сообщений, мультимедиа только в специальных режимах
    • Капча: активируется при превышении лимита запросов (5 за 10 секунд), генерирует математические задачи

Купить подписку

Выбор периода подписки и оплата через Telegram Payments.

  1. В меню "Подписка на бота" выбрать "Купить подписку"

  2. В меню "Ваши подписки" выберите:
    • Посмотреть подписки
    • История подписок
ЦЕНТР ПОДПИСОК

Посмотреть подписки

Просмотр активных подписок, их сроков и стоимости.

Инструкция по использованию:

  1. В меню "Ваши подписки" выбрать "Посмотреть подписки"
  2. Просмотреть список активных подписок и итоговую оценку
ЦЕНТР ПОДПИСОК

История подписок

Калькулятор предназначен для рассчета количества потраченного топлива и средств на поездку по заданному маршруту.

Инструкция по использованию:

  1. В главном меню выберите "Калькуляторы"
  2. В меню "Калькуляторы" выберите "Расход топлива"
  3. Ввод маршрута:
    • Укажите начальное местоположение (через геолокацию или текст)
    • Укажите конечное местоположение
    • При необходимости добавьте дополнительные точки маршрута
  4. Расстояние:
    • Выберите "Автоматическое" для расчета расстояния по геолокации
    • Или выберите "Ввод вручную" и укажите расстояние в километрах
  5. Дата поездки:
    • Введите дату в формате или выберите "Пропустить ввод даты"
  6. Тип топлива:
    • Выберите тип топлива из списка: АИ-92, АИ-95, АИ-98, АИ-100, ДТ, ГАЗ
  7. Цена топлива:
    • Выберите "Актуальная цена" для средней цены по данным, если доступно
    • Или выберите "Ввести свою" и укажите цену за литр в рублях
  8. Расход топлива:
    • Укажите расход топлива на 100 км в литрах
  9. Количество пассажиров:
    • Введите количество пассажиров

Результат:

Бот выдает отчет с:

  • Местоположений маршрута
  • Даты поездки
  • Расстояния
  • Типа топлива и его цены
  • Расхода топлива
  • Количества пассажиров
  • Общего расхода топлива (в литрах)
  • Общей стоимости поездки
  • Стоимости на одного пассажира
  • Ссылки на маршрут в Яндекс.Картах

Просмотр и удаление поездок:

  • Выберите "Посмотреть поездки" для просмотра (есть возможность просмотра в excel)

      • Медовуха, Наливка, Настойка, Абсент, Бренди, Вермут, Вино игристое
      • Вино крепленое, Вино некрепкое, Виски, Водка, Джин, Квас, Пиво
      • Пиво безалкогольное, Пиво крепкое, Самбука, Ром, Сидр, Текила, Тоник
    • Объем напитков:
      • л.
      • мл.
    • Крепость напитков (%)
    • Скорость употребления:
      • Сразу
      • За 1-10 часов
    • Время с последнего приема:
      • Только что
      • Ввод своего времени в часах или минутах
    • Наличие еды:
      • Нет
      • Закуски

    • Пол:
      • Мужчины: 0.7
      • Женщины: 0.6
    • Скорость:
      • Сразу: -0%
      • За 1-10 часов: -n%

      • Нет: -0%
      • Закуски: -10%
      • Пища: -30%

  3. Выводит результат:
    • Текущий уровень алкоголя (%)
    • Время до полного вытрезвления
    • Дата и время вытрезвления
    • Рекомендации:
      • 0.0-0.3 промилле: слабое опьянение

Примечание:

Формулы расчета:

  • C = A / W * r, где:
    • C - концентрация алкоголя в крови (промилле)
    • A - масса чистого алкоголя в граммах (V * 1000 * S * 0.79)
    • W - вес человека в кг.
    • r - коэффициент распределения (0.7 для мужчин, 0.6 для женщин)
    • V - объем напитка в литрах
    • S - крепость напитка в долях
    • p (этилового спирта) - 0.79

Важная информация:

  • Калькулятор не сможет дать 100% точный результат! Если вы выпили, то НИ в коем случае нельзя садиться за руль после алкоголя как минимум в течение суток!!!
  • Уровень алкоголя в крови зависит от множества индивидуальных факторов (метаболизм, состояние здоровья, прием лекарств)
  • Снижение концентрации за счет еды и скорости употребления является приблизительным
  • Даже малые дозы алкоголя могут влиять на скорость реакции и внимание
  • В РФ допустимая норма алкоголя за рулем составляет 0.3 промилле (погрешность измерения)

    • Проценты начисляются на остаток

1. Сбор и использование данных

1.1. Категории данных

Бот собирает следующие данные:

  • Telegram данные: ID пользователя, username, имя (для идентификации и связи)
  • Данные подписок: Информация о покупках, активных подписках, истории платежей
  • Данные калькуляторов: Параметры для расчетов (пробег, тип топлива, данные автомобиля, личные параметры для калькулятора алкоголя)
  • Транспорт и расходы: Информация о добавленных автомобилях, записи трат и ремонтов
  • Геолокация: Координаты для поиска мест, антирадара, погоды (обрабатываются и не сохраняются долгосрочно)
  • Напоминания: Тексты задач, даты, настройки повторений
  • Настройки: Параметры уведомлений, предпочтения пользователя
  • Коммуникации: Сообщения в чате с администратором, рекламные заявки
  • Системные данные: Логи активности, статистика использования функций (в обезличенном виде)

1.2. Цели обработки

Данные используются для:

  • Предоставления всех функций бота (калькуляторы, учет расходов, поиск, уведомления)
  • Управления подписками и обработки платежей
  • Персонализации услуг (напоминания, настройки уведомлений)
  • Обеспечения безопасности (защита от спама, мониторинг активности)
  • Технической поддержки через чат с администратором
  • Улучшения функционала на основе анонимной статистики
  • Отправки системных уведомлений о состоянии подписки

1.3. Источники данных

Данные поступают от:

  • Прямого ввода пользователем в боте
  • Telegram API (базовая информация профиля)
  • Геолокации, отправляемой пользователем
  • Платежных систем (подтверждения транзакций)
  • Сторонних сервисов при выполнении запросов:
    • Яндекс.Карты (поиск мест, маршруты)
    • OpenWeatherMap, Open-Meteo, WeatherAPI (погода)
    • AvtoCod (проверка автомобилей)
    • Fuelprice, Petrolplus (цены на топливо)
    • ЦБ РФ, MOEX (курсы валют)

2.1. Методы обработки

Данные обрабатываются автоматизированно:

  • Сбор через Telegram API и пользовательский ввод
  • Хранение в защищенных базах данных в РФ
  • Обработка для выполнения запросов (расчеты, поиск)
  • Передача минимально необходимых данных сторонним сервисам
  • Удаление по истечении сроков или по запросу

2.2. Принципы

  • Законность: Обработка только с согласия пользователя
  • Минимизация: Сбор только необходимых данных
  • Прозрачность: Открытая информация о целях обработки
